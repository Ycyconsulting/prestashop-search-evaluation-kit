# Contributing

Open an issue to propose one observable search scenario at a time.

Include:

- the PrestaShop major version and general search architecture;
- a minimal synthetic fixture;
- the user action;
- the expected observation;
- why an existing checklist row does not cover it.

Do not include credentials, private store URLs, customer or order data, production logs, license keys, or a vendor accusation. Reproduce security issues privately with the affected vendor before creating a public issue.

Changes must remain vendor-neutral and must not turn a fixture observation into a performance, compatibility, legal, or revenue guarantee.
